# PS Building - Next.js Rebuild

This is a modern rebuild of the PS Building website using Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `/app`: App Router pages and layouts.
- `/components`: Reusable UI components.
- `/lib`: Utility functions.
- `/public`: Static assets (images, fonts).

## Features

- **Responsive Design:** Fully responsive for mobile, tablet, and desktop.
- **Dynamic Gallery:** Project categories are generated dynamically.
- **Interactive UI:** Custom carousels, lightboxes, and smooth animations.
- **SEO Friendly:** Server-side rendering and semantic HTML.
